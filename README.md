# Qibla

تطبيق Android بسيط لتحديد اتجاه القبلة باستخدام الموقع وحساسات الجهاز.

يوفر التطبيق بوصلة للقبلة وخريطة مساعدة، مع إمكانية اختيار الموقع يدويًا عند ضعف GPS أو عند الاستخدام بدون إنترنت.

## المميزات

- تحديد اتجاه القبلة بالبوصلة.
- عرض اتجاه القبلة على الخريطة.
- دعم اختيار الموقع يدويًا عند ضعف GPS.
- يعمل حساب اتجاه القبلة بدون إنترنت إذا كان الموقع معروفًا.
- إعدادات لتغيير حجم الخط ونوع الخط.
- دعم خط القاهرة.
- عرض معلومات التطبيق والإصدار وحقوق البرمجة داخل التطبيق.

## الصلاحيات

- الموقع: لحساب اتجاه القبلة من مكان المستخدم.
- الإنترنت وحالة الشبكة: لتحميل الخريطة عند استخدام وضع الخريطة.
- حساسات الجهاز: قراءة اتجاه الهاتف مثل البوصلة.
